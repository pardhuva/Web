
//colors & background 

  | Prefix    | Meaning                               | Example                      |
| :-------- | :------------------------------------ | :--------------------------- |
| `bg-`     | background color                      | `bg-blue-500`, `bg-gray-100` |
| `text-`   | text color                            | `text-red-600`, `text-white` |
| `border-` | border color                          | `border-green-400`           |
| `divide-` | divider color (for child elements)    | `divide-gray-300`            |
| `accent-` | accent color (for inputs, checkboxes) | `accent-pink-500`            |

//spacing 
 
  /margin 

  | Class Prefix | Applies To       | Example         | Meaning                               |
| :----------- | :--------------- | :-------------- | :------------------------------------ |
| `m-`         | all sides        | `m-4`           | margin on all 4 sides                 |
| `mt-`        | top              | `mt-4`          | margin-top                            |
| `mb-`        | bottom           | `mb-4`          | margin-bottom                         |
| `ml-`        | left             | `ml-4`          | margin-left                           |
| `mr-`        | right            | `mr-4`          | margin-right                          |
| `mx-`        | left & right     | `mx-4`          | horizontal margins                    |
| `my-`        | top & bottom     | `my-4`          | vertical margins                      |
| `-m-`        | negative margin  | `-m-2`, `-mt-1` | pulls element closer (negative space) |
| `auto`       | automatic margin | `mx-auto`       | centers element horizontally          |

 /padding 
  | Class Prefix | Applies To   | Example | Meaning                |
| :----------- | :----------- | :------ | :--------------------- |
| `p-`         | all sides    | `p-4`   | padding on all 4 sides |
| `pt-`        | top          | `pt-4`  | padding-top            |
| `pb-`        | bottom       | `pb-4`  | padding-bottom         |
| `pl-`        | left         | `pl-4`  | padding-left           |
| `pr-`        | right        | `pr-4`  | padding-right          |
| `px-`        | left & right | `px-4`  | horizontal padding     |
| `py-`        | top & bottom | `py-4`  | vertical padding       |

  you know the difference right margin is from the external elements whereas padding is inside element with the text

//flexbox & grid

    | `grid`, `inline-grid` | display grid                  | `grid-cols-3`, `grid-rows-2`        |

    | `items-`              | cross-axis alignment          | `items-center`, `items-start`       |
